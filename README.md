# Programacao_1_Cpp

This repository contains exercises and projects developed during the **Programming I (C++)** course at **IFC (Instituto Federal Catarinense)**. The content focuses on object-oriented programming fundamentals and practical applications using C++.

## üìö Repository Structure

The repository is organized by class sessions (Aulas), with each folder containing exercises that progressively build programming skills:

### **Aula_01 - Fundamentals & Basic Classes**
- **ex001**: Circle class - area and perimeter calculations
- **ex002**: Bank account system - deposits, withdrawals, and balance management
- **ex003**: Rectangle class - geometric calculations
- **ex004**: Student class - grade management and approval status
- **ex005**: Employee class - salary calculations with taxes and benefits
- **ex006**: Advanced employee system with salary increases and annual earnings

### **Aula_02 - Inheritance & Complex Systems**
- **ex005**: Library management system with user authentication, book lending, and admin privileges using inheritance and multiple classes (User, Livro, Emprestimo, Admin)
